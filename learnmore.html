Number System Conversion

As we know, the number system is a form of expressing numbers. In number system conversion, we will study how to convert a number from one base to another base. There are a variety of number systems such as binary, decimal, hexadecimal, and octal, which can be exercised.

In this article, you will learn the conversion of one base number to another base number, considering all the base numbers such as decimal, binary, octal, and hexadecimal, with the help of examples. Below are the methods for converting number systems:

  • Binary to Decimal Number System
  • Decimal to Binary Number System
  • Octal to Binary Number System
  • Binary to Octal Number System
  • Binary to Hexadecimal Number System
  • Hexadecimal to Binary Number System

Get the PDF of the number system with a brief description in it. The general representation of number systems are as follows:

  • Decimal Number – Base 10 – N10
  • Binary Number – Base 2 – N2
  • Octal Number – Base 8 – N8
  • Hexadecimal Number – Base 16 – N16
